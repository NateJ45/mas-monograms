# MAS Monograms — logo concepts (2026-07-02)

Status: **decided and implemented 2026-07-02.** Five directions were designed against real-brand
research; Nate shortlisted #2 and #3 (flagging the wide-tracked "M A S" deck, removed in the refined
versions saved here), then chose the **hybrid system**: the Flourished Initial lockup lives in
`src/components/Logo.astro` (header + footer), and the Shopkeeper's Badge is the compact mark —
`<Logo mark />`, `public/favicon.svg`, and the app-icon set generated by
`scripts/generate-favicons.mjs`. The files in this folder are the design-reference versions
(including the tagline microlines that were dropped from the production lockup).

## Research grounding (real sites, opened and verified)

- Mass-market monogram shops (Marleylilly, United Monograms, Mark & Graham, Initial Outfitters,
  South of Hampton): the mark is almost always the brand's own initials, and the favicon is always
  the detached initials-in-a-circle — never the wordmark.
- Heritage/luxury embroidery brands (Leontine Linens, Matouk, Weezie, Courtland & Co.): letterspaced
  serif caps in one muted ink; script rationed to a single element (usually one initial); place-name
  microtext in the lockup; tool imagery (needles/thread) absent from the primary mark; responsive
  logo *families* (full lockup → reduced lockup → bold mark) rather than naive shrinking.
- Favicon best practice (Evil Martians et al.): ship favicon.ico 32px + icon.svg + apple-touch-icon
  180px (solid bg, ~20px padding) + icon-192/512 + manifest; ≥2px strokes at 16px; thin scripts
  never survive tab size.

## Finalists

### 2 — The Shopkeeper's Badge (`2-shopkeepers-badge/`)
Double indigo ring reading as an embroidery hoop (screw-tab detail at top), Claret Fraunces "M"
center. Stacked form arcs "MAS MONOGRAMS" / "ST. MATTHEWS, SC" around the rim. Mark = rings + M,
no text. At a dedicated 16px cut: drop the inner ring, thicken the outer to 8 units.

### 3 — The Flourished Initial (`3-flourished-initial/`)
Oversized Petemoss script "M" in Claret with a drawn thread-swash underlining a normal-tracked
Fraunces "MAS MONOGRAMS". Mark deliberately swaps the script for a bold Fraunces M + thick claret
swash (the luxury-brand favicon reduction — script never shrinks below ~48px).

The two share the claret M + indigo/ink palette, so a **hybrid system** is coherent: flourished
lockup in header/footer text contexts, badge as the favicon/social/stamp mark (the Weezie pattern:
wordmark + separate mark + initial favicon).

## Production notes (for whichever wins)

- These SVGs use live `<text>`; they render correctly on the site because Fraunces/Mulish/Petemoss
  are loaded there. Before shipping as standalone assets (favicon.svg, og images, print), convert
  text to outlined paths (e.g. opentype.js against the Fontsource TTFs in node_modules) — a favicon
  SVG cannot load webfonts.
- `public/favicon.svg` is still the OLD pre-Heirloom-Coast mark (old teal/rust hexes) and must be
  replaced regardless of the chosen direction.
- New Logo.astro should keep the `variant` / `mark` prop API so Header/Footer call sites don't change.
